# Friends — AI Virtual Pet Companion 🐾

A mobile-first AI virtual pet app where you adopt, raise, and grow a pet with a friend.

## Live Demo
🔗 https://friends-ai-pet.vercel.app

## Features
- 🐱 Pet creation with name, species and personality
- 🍖 Daily care actions (Feed, Play, Sleep, Talk)
- 🤖 AI personality engine powered by Groq LLaMA 3.1
- 👫 Co-parenting — share your pet with a friend
- 🌱 Growth stages with visual evolution (6 stages)
- 📖 Pet journal with milestone memories
- ✨ Premium animations and effects

## Tech Stack
- React Native + Expo (Web)
- Supabase (Auth + Database)
- Groq API (LLaMA 3.1-8b-instant)
- Vercel (Deployment)

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Add your keys to `.env`
4. Run `npx expo start --web`
